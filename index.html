Dataset Description:

We analyzed information from datasets in Kaggle, Yahoo Finance, and Bloomberg, with Yahoo Finance being the most popular dataset for financial data [4]. Through our analysis, we decided to use the Kaggle "Stock Market Dataset" which offers more than two decades of information on stocks and ETFs. The dataset includes the following content for each stock and ETF on a daily basis: date, opening price, closing price, lowest price, highest price, adjusted closing price, and trading volume. To narrow the focus of our model, we analyze ten of the most prevalent and historic stocks: Apple, Amazon, Google, Microsoft, Nvidia, Tesla, Meta, Berkshire Hathaway, JPMorgan, and Disney.

Problem/Motivation:

The problem we are trying to solve is the difficulty of predicting financial markets. With financial institutions having more resources at their disposal, we aim to use machine learning algorithms to help everyday traders achieve their financial goals.

Methods:

Data preprocessing:

  1. Normalization: We are going to find the means, standard deviations, open/closing prices of the stocks we have chosen.
  2. Principal Component Analysis: Use Singular value decomposition to define principal components within our datasets.
  3. Cross Validation Sets: Break up our components into training and validations sets by using Kaggle and Yahoo Finance to compare wih the Bloomberg data.

Models:

  1. Linear Regression: Comparing variables.
  2. Support Vector Machine: Separating the data into different categories.
  3. Neural Networks: Use Long Short Term Memory (LSTM) to find long term dependencies.

(Potential) Results and Discussion:

Labeling Predictions:

  • True Positive (TP): Model predicts an uptick. There is an uptick.
  • True Negative (TN): Model predicts no uptick. There is no uptick.
  • False Positive (FP): Model predicts an uptick. There is no uptick.
  • False Negative (FN): Model predicts no uptick. There is an uptick.

Metrics:

  • Accuracy: (TP + TN) / Total - Correctness
  • Precision: TP / (TP + FP) - Quality
  • Recall: TP / (TP + FN) - Completeness

Project Goals:

The objective of our model is to predict whether a stock will have a higher closing price compared to opening price on a given day, which we define as an uptick.

Expected Results:

The Long Short Term Memory (LSTM) model is expected to have the most optimal predictions of the market. This is because LSTM networks "capture the context of the data as they are being trained and can address the gradient vanishing problem, making them the idea approach for time series forecasting [4]." The Linear Regression model is expected to have the least accuracy in accordance to our metrics as stock movements have a "non linear, dynamic and complex nature [3]" which cannot be robustly represnted by a simple regression model. Our prediction is backed by several tests as past surveys on papers indicate "it is evident that LSTM is the most preferred model among researchers for predicting stock price movements [5]." The SVM model is predicted to have a reasonable accuracy, but has some drawbacks related to the "selection of the kernel function parameters [...] and the value in the loss insensitive function [3]".
