Problem/Motivation:

The problem we are trying to solve is the difficulty of predicting financial markets. With financial institutions having more resources at their disposal, we aim to use machine learning algorithms to help everyday traders and day traders achieve their financial goals.

Methods:

Data preprocessing:

  1. Normalization: We are going to find the means, standard deviations, open/closing prices of the stocks we have chosen.
  2. Principal Component Analysis: Use Singular value decomposition to define principal components within our datasets.
  3. Cross Validation Sets: Break up our components into training and validations sets by using Kaggle and Yahoo Finance to compare wih the Bloomberg data.

Models:

  1. Linear Regression: Comparing variables.
  2. Support Vector Machine (Supervised): Separating the data into different categories.
  3. Neural Networks: Use Long Short Term Memory to find long term dependencies.

(Potential) Results and Discussion:

Metrics:

  • Accuracy (correct prediction / total predictions) - Prediction reliability
  • Precision (correct “uptick” prediction / total “uptick” prediction) - Measures true/false positives
  • Recall (correct “uptick” prediction / actual “uptick” predictions) - Ability to determine upticks
